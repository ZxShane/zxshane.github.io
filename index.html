Research

2026 - Present

Active Memory for Days-long Video Agents

Developing an Active Explore Memory Agent for days-level egocentric video understanding. The agent actively writes, retrieves, compresses, and updates memory as it explores long videos.

5.3× faster memory retrieval and up to 6.1× memory compression on EgoLifeQA and Ego-R1 Bench.

2026 - Present

Resource-efficient Reinforcement Learning for Long-horizon Agents

Studying low-resource RL training for long-trajectory terminal agents using parameter sharding, parallel rollouts, domain-specific data generation, and sparse-reward optimization.

Completed 9B-scale long-trajectory agent RL training with six GPUs.

2025

EmoOmni: Omni-modal Emotion Reasoning

Proposed the Omni-CoT data construction framework and DyME-GRPO, a reinforcement learning method that balances the use of visual, audio, and textual evidence in omni-modal models.

ACL 2026 Findings · ICASSP 2026 HumDial Challenge global third place.

2020 - 2022

Adversarial Privacy-preserving Filters

Designed transferable adversarial filters for portrait privacy protection and built a real-time mobile inference system using InsightFace, React Native, and Flask.

Published at ACM Multimedia and IEEE Transactions on Multimedia.
