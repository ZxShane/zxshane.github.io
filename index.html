研究经历

2026 - 至今

面向 Days-level 超长视频的主动记忆 Agent

面向第一视角超长视频理解任务，研究 Agent 的长期记忆形成、动态检索与压缩问题。提出 Active Explore Memory Agent，在探索长视频的过程中主动写入、检索、压缩并更新记忆。

在 EgoLifeQA 和 Ego-R1 Bench 上实现 5.3× 更快的记忆检索，以及最高 6.1× 的记忆压缩。

2026 - 至今

面向长程 Agent 的低资源强化学习系统优化

针对长轨迹 Terminal Agent，结合参数分片、并行 Rollout、领域数据生成和稀疏奖励优化，研究有限计算资源下的大模型强化学习训练方法。

在 6 张 GPU 上完成 9B 规模长轨迹 Agent 的强化学习训练。

2025

EmoOmni：全模态情绪推理与模态均衡强化学习

提出 Omni-CoT 多模态推理数据构建范式和 DyME-GRPO 强化学习方法，改善 Omni 模型在视觉、音频和文本信息利用上的失衡问题。

ACL 2026 Findings · ICASSP 2026 HumDial Challenge 全球第三名。

2020 - 2022

对抗攻击分析与隐私保护滤镜

设计具备迁移能力的人像隐私保护对抗滤镜，并基于 InsightFace、React Native 和 Flask 构建移动端实时推理系统。
